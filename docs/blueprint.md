# **App Name**: RoadRunner Deliver

## Core Features:

- Onboarding & Identity Verification: Streamlined multi-step process for new delivery partners to register, verify phone number with OTP, upload KYC documents (DL, RC, Insurance, Aadhaar), and view their verification status (pending, approved, rejected).
- Online/Offline Availability Management: Riders can easily toggle their availability status between online and offline. When online, their real-time location is tracked for order assignment, adhering to active GPS and service area rules.
- Real-time Order Assignment & Response: Receive push notifications for new delivery orders with an immediate, full-screen popup. Riders can view order details and accept or reject within a set timer, triggering re-assignment logic if no response.
- Guided Delivery Workflow & Navigation: Step-by-step guidance for active orders, from navigating to pickup and drop-off locations (with integrated Google Maps routes and external map app option) to marking arrival, pickup, and OTP-verified delivery.
- Earnings & Incentives Dashboard: Provides a clear overview of daily and weekly earnings, a detailed list of completed orders with associated amounts, and tracks progress towards various incentive goals with a visual progress bar.
- AI-Powered Delivery Guidance Tool: A generative AI tool that assists riders by clarifying ambiguous customer delivery instructions, providing context, or suggesting questions to ensure accurate and smooth deliveries.
- Personal Profile & Support Access: Riders can manage their profile details, update vehicle information, review their uploaded documents, and access quick support options including calling support or submitting an issue report.

## Style Guidelines:

- Primary color: A professional and trustworthy blue (#2675C4) for main branding and active elements.
- Background color: A soft, desaturated light blue-gray (#ECF1F6) to provide a clean and calm canvas.
- Accent color: A vibrant purple (#8040FF) for distinct interactive elements, alerts, and key highlights.
- Action colors: Specific colors for functionality: Green (#4CAF50) for accept/online, Red (#F44336) for reject/alerts, and Yellow (#FFC107) for incentives.
- All text: 'Inter', a modern sans-serif, for exceptional readability and a clean, efficient aesthetic suitable for both headlines and body text, ensuring fast information consumption.
- Utilize bold, clear, and instantly recognizable icons to support rapid visual interpretation. Icons should be sized generously for thumb-friendly interaction.
- Adopt a minimal and intuitive layout, prioritizing crucial information. Implement large, easily tappable buttons, with primary actions consistently positioned at the bottom of the screen for single-hand usability.
- Integrate subtle, smooth transitions and visual feedback for state changes (e.g., online/offline toggle, order acceptance) to enhance user experience without causing delays.
